CGMBot

Your Discord Bot for Dexcom Glucose Monitoring

About CGMBot

Monitor your glucose levels in real-time using Dexcom data directly from Discord.

Features

  • Securely set your Dexcom credentials.
  • Configure custom glucose alerts.
  • Receive real-time glucose readings.
  • Get notified when glucose levels are out of range.

Commands

Set Dexcom Credentials setcredentials <username> <password>
Set Glucose Limits setlimits <lower> <upper>
Get Current Glucose getglucose
View Stored Data getdata
Remove Data removedata <credentials|limits|all>

Installation

Invite CGMBot to your Discord server and follow the instructions above.

Check out the GitHub repository for more details.
